# Skills

This directory contains topic-based Google Ads skills for Codex and Claude Code.

## How To Use

- Start with the skill that matches the problem domain.
- Use the `docs_range` in `skills-manifest.json` to jump to the right topic cluster.
- Keep executable API work in the CLI.
- Keep policy, troubleshooting, and process guidance in skills.

## Topic Map

- `google-ads-auth` for auth, OAuth, service accounts, and access setup
- `google-ads-query` for GAQL, reporting, and search surfaces
- `google-ads-mutate` for writes, partial failures, and batch jobs
- `google-ads-campaigns` for campaign setup across campaign types
- `google-ads-experiments` for experiment lifecycle work
- `google-ads-conversions` for conversions, uploads, billing, and forecasting
- `google-ads-assets` for assets and creative generation
- `google-ads-account-management` for account linking and hierarchy
- `google-ads-planning` for keyword planning and insights
- `google-ads-productionize` for release, upgrade, and operational guidance
- `google-ads-troubleshooting` for errors, policy, and best practices
- `google-ads-remarketing` for audiences and user lists
- `google-ads-verticals` for hotel, travel, and recommendations flows
